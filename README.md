# Hesamoddin Jafari - Portfolio & CV Website

A modern, interactive personal portfolio website featuring Apple-inspired UI design, custom momentum scrolling, and interactive project showcases.

## 🚀 Quick Start (Run Locally)

### 1. Clone the repository
```bash
git clone https://github.com/hjafari890/hesam-cv.git
cd hesam-cv
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run development server
```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🛠️ Built With
- **React** + **Vite**
- **Framer Motion**
- **Lucide React**
- **Vanilla CSS**
